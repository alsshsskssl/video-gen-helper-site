# 개인정보처리방침

**최종 수정일**: 2025년 3월 15일

Video Gen Helper(이하 "서비스")의 개인정보처리방침입니다.

## 1. 수집하는 정보

본 서비스는 YouTube 관련 기능 제공을 위해 다음 정보에 접근합니다:

- **YouTube 채널 정보**: 채널명, 채널 ID
- **YouTube 업로드 권한**: 사용자가 직접 영상을 업로드하기 위한 OAuth 인증 토큰
- **YouTube 분석 데이터**: 조회수, 시청 시간, 구독자 변화 등 채널 분석 정보

## 2. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

- 사용자가 제작한 영상을 본인의 YouTube 채널에 업로드
- 영상 썸네일 설정
- 채널 분석 데이터 조회 및 표시

## 3. 정보 저장 방식

- OAuth 인증 토큰은 **사용자의 로컬 환경(본인 컴퓨터)**에만 저장됩니다.
- 외부 서버에 사용자 인증 정보를 전송하거나 저장하지 않습니다.
- 사용자는 언제든지 로컬에 저장된 토큰 파일을 삭제하여 인증을 해제할 수 있습니다.

## 4. 제3자 정보 제공

본 서비스는 수집된 사용자 정보를 **제3자에게 공유, 판매, 제공하지 않습니다.**

## 5. Google API 서비스 사용

본 서비스는 Google API 서비스를 사용하며, [Google API 서비스 이용약관](https://developers.google.com/terms)을 준수합니다. 또한 본 서비스의 Google API를 통해 수신한 정보의 사용 및 다른 앱으로의 전송은 [Google API 서비스 사용자 데이터 정책](https://developers.google.com/terms/api-services-user-data-policy)을 준수합니다.

## 6. 데이터 보안

- 모든 인증 정보는 사용자 로컬 환경에서만 관리됩니다.
- Google OAuth 2.0 표준 인증 프로토콜을 사용합니다.
- HTTPS를 통한 암호화된 통신만 사용합니다.

## 7. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

- **인증 해제**: 로컬 토큰 파일 삭제 또는 [Google 계정 설정](https://myaccount.google.com/permissions)에서 앱 접근 권한 취소
- **데이터 삭제**: 로컬에 저장된 모든 프로젝트 데이터를 직접 삭제 가능
